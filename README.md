## 🌟 About The Project

**Crunk Games** is a revolutionary gaming platform that combines competitive gaming, social interaction, and real rewards into one seamless experience. Built by passionate young developers, our platform allows gamers to create leagues, compete in tournaments, earn Veno Coins, and connect with friends — all in one place.

Whether you're an eFootball enthusiast, EA Sports FC pro, or DLS master, Crunk Games provides the perfect arena to showcase your skills and earn rewards!

### 🎯 Our Mission
To create the most engaging and rewarding gaming platform where every player can:
- Compete fairly in organized tournaments
- Build and manage their dream teams
- Connect with fellow gamers worldwide
- Earn real value through their gaming skills

## ✨ Features

### 🏆 **Tournament & League System**
- Create custom leagues with flexible formats (Round Robin, Group Stage, Knockout)
- Set entry fees and prize pools
- Real-time standings and match fixtures
- Match result verification with screenshot uploads
- Owner dashboard for league management
- Team approval system

### 🎮 **Team Management**
- Create your dream team with custom logos
- Build full squads with player ratings (GK, RB, CB, LB, CM, RW, LW, ST)
- Choose formations (4-3-3, 4-4-2, 3-5-2, 5-3-2, 4-2-3-1)
- Track team statistics and performance

### 💰 **Veno Coins Economy**
- Earn coins through daily rewards (10 coins every 24 hours)
- Participate in paid tournaments to win big
- Win coins through spin wheel (50 - 5000 coins)
- Complete achievements for bonus coins

### 🎰 **Spin & Win**
- Exciting prize wheel with up to 5000 coin rewards
- Visual spinning animation
- Confetti effects for big wins
- Prize odds system

### 👤 **User Features**
- Google Authentication for secure login
- Personalized profiles with gaming statistics
- Dark/Light theme support
- Mobile-responsive design
- Notification system

## 💬 Integration with QONVO App

### What is QONVO?
**QONVO** is our revolutionary chat application that powers real-time communication within Crunk Games. It provides seamless messaging between players, league chat rooms, and match coordination.

### 🔗 QONVO Features in Crunk Games
- **Auto-login** - No need to sign in separately
- **League-specific chat rooms** - Discuss strategies with teammates
- **Match coordination** - Schedule and organize matches
- **Friend messaging** - Private conversations
- **Real-time notifications** - Never miss a match update
- **Group chats** - Create team chat groups

### 📱 Try QONVO
Experience the power of seamless gaming communication:
- **Website:** [QONVO Chat App](https://qonvo-chat.netlify.app/)
- **Features:** Real-time messaging, group chats, file sharing
- **Integration:** Automatically connects with your Crunk Games account
